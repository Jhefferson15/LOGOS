# Logos - O Jogo da Filosofia

## Descrição

Logos é um jogo interativo que explora o mundo da filosofia, permitindo aos jogadores debater ideias, aprender com grandes pensadores e mergulhar em diferentes escolas filosóficas. O jogo é projetado para ser uma experiência educacional e divertida, combinando elementos de RPG com desafios intelectuais.

## Como Rodar o Projeto

### Pré-requisitos

Certifique-se de ter o Node.js e npm (ou Yarn) instalados para o frontend, e Python (com pip) para o backend.

### Frontend

1.  Navegue até o diretório `Frontend`:
    ```bash
    cd Frontend
    ```
2.  Abra o arquivo `index.html` em seu navegador ou use um servidor web local (por exemplo, Live Server para VS Code).

### Backend

(Assumindo que o backend é baseado em Python. Adapte conforme a tecnologia real do backend)

1.  Navegue até o diretório `Backend`:
    ```bash
    cd Backend
    ```
2.  Instale as dependências (se houver):
    ```bash
    pip install -r requirements.txt
    ```
3.  Execute o servidor backend:
    ```bash
    python app.py # ou o comando de inicialização do seu backend
    ```

## Como Instalar como PWA (Progressive Web App)

Para instalar o Logos como um aplicativo em seu dispositivo (especialmente em navegadores móveis como o Chrome):

1.  Abra o jogo no seu navegador (certifique-se de que está acessando via `http://` ou `https://`).
2.  No Chrome (Android), você verá um banner "Adicionar Logos à tela inicial" ou poderá encontrar a opção "Instalar aplicativo" no menu do navegador (geralmente os três pontos verticais).
3.  Clique em "Instalar" ou "Adicionar à tela inicial" para instalar o jogo como um PWA.
