concepts.js Este arquivo define o banco de dados dos conceitos filosóficos. Cada conceito possui um nome, uma breve descrição para fins de jogo, uma pontuação associada e uma lista de filósofos que o desenvolveram. Este é o principal recurso que os jogadores usam para pontuar, conectando o conceito correto ao filósofo em jogo.
