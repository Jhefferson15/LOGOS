## Objects

<dl>
<dt><a href="#Data">Data</a> : <code>object</code></dt>
<dd><p>arenas.js</p>
<p>Define as arenas de batalha do jogo. Cada arena representa uma era ou um
grande movimento filosófico, servindo como &quot;lar&quot; para um conjunto específico
de escolas de pensamento. A progressão pelas arenas guia o jogador através
da história da filosofia.</p>
</dd>
<dt><a href="#Data">Data</a> : <code>object</code></dt>
<dd><p>concepts.js</p>
<p>Este arquivo define o banco de dados dos conceitos filosóficos.
Cada conceito possui um nome, uma breve descrição para fins de jogo,
uma pontuação associada e uma lista de filósofos que o desenvolveram.
Este é o principal recurso que os jogadores usam para pontuar, conectando
o conceito correto ao filósofo em jogo.</p>
</dd>
<dt><a href="#Data">Data</a> : <code>object</code></dt>
<dd></dd>
<dt><a href="#Data">Data</a> : <code>object</code></dt>
<dd><p>philosophers.js</p>
<p>Este arquivo contém o banco de dados principal dos filósofos para o jogo.
Cada filósofo é um objeto com informações essenciais como data de nascimento,
escola filosófica, conceitos-chave e predecessores, permitindo a criação de
mecânicas de jogo complexas como linhas de pensamento e evolução conceitual.</p>
<p><strong>MODIFICADO:</strong> Adicionada a propriedade <code>pos</code> a cada filósofo para layout visual.</p>
